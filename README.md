# fileDownloader
Паралельний завантажувач файлів, з використанням горутин для одночасного завантаження файлів з різних URL-адрес. 
# Функціонал
- Завантаження кількох файлів паралельно
- Вивід інформації про роботу кожного воркера
- Вивід статистики
  
# Прикола роботи
```bash
go run main.go
```
